Descripción general:
Este proyecto presenta un análisis ejecutivo de las ventas de Walmart con el objetivo de identificar los tipos de tienda más eficientes, los departamentos con mayor y menor contribución al negocio, y los principales riesgos de volatilidad de ventas durante el año 2012.
Objetivo:
•	Identificar qué tipos de tienda generaron mayores ventas.
•	Determinar qué departamentos estuvieron por encima y por debajo del promedio.
•	Analizar la volatilidad de las ventas y los posibles riesgos asociados.
Herramientas y tecnologías:
•	Google Sheets
•	Funciones avanzadas
•	Tablas dinámicas
•	Dashboards
Metodología:
•	Limpieza y preparación de datos.
•	Agregación y análisis comparativo de ventas.
•	Creación de tablas dinámicas y dashboards para reporting ejecutivo.
Principales hallazgos:
•	Algunos tipos de tienda mostraron un desempeño superior en términos de ventas totales.
•	Se identificaron departamentos con bajo aporte al negocio.
•	La volatilidad de ventas representó un riesgo relevante en ciertos períodos del año.
Aprendizajes clave:
•	Análisis de datos orientado a la toma de decisiones ejecutivas.
•	Visualización de datos para comunicar insights de forma clara.
•	Identificación de riesgos y oportunidades mediante análisis descriptivo.
